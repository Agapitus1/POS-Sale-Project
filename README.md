# CAN-DUMP-Webapp

IAG Webapp 2.0
This is the second version of the web application, using MongoDB database, Node & Express for backend, and React for frontend

Install node packages
cd /backend
npm i
cd ../frontend
npm i
Run backend
From repo root folder:

cd /backend
npm run dev
Run frontend
From repo root folder

cd /frontend
npm start
From /backend

cd ../frontend
npm start
